01 Integration

Plivo + EMAAVY architecture

Plivo excels at cost-efficient international termination. EMAAVY's Plivo adapter normalizes call events into the same call_id schema used for Twilio — swap providers per campaign without rewriting agent logic.

Media streams use Plivo's AudioStream API with automatic codec negotiation and packet loss concealment tuned for LLM turn-taking.
