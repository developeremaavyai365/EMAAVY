01 Metrics

What you see in real time

EMAAVY's dashboard streams telemetry from the inference plane and telephony edge:

  • Active calls map — live transcript preview and sentiment color coding per session
  • Campaign funnel — connected, qualified, booked, and lost outcomes updated per dial
  • Agent leaderboard — conversion rate, average handle time, tool success ratio
  • System latency — STT, LLM, TTS, and tool execution P50/P95/P99 with SLA breach alerts
02 Reporting

Export & BI hooks

Schedule daily CSV exports to S3 or push metrics to your warehouse via webhook. Every chart maps to a GET /v1/analytics/* endpoint for programmatic reporting.
