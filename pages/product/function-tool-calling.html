Product Features

Function & Tool Calling

Define OpenAI-compatible function schemas that agents invoke mid-call — CRM lookups, payment links, calendar bookings — executed in EMAAVY's audited sandbox with sub-250ms p95.

01 Execution

Secure tool runtime

Tools never run on the LLM provider's infrastructure. EMAAVY validates JSON arguments against your schema, executes HTTP webhooks or native integrations (Salesforce, HubSpot, Cal.com), and injects results back into the conversational context before the next TTS chunk synthesizes.

Failed tools trigger graceful verbal fallbacks — agents apologize, retry once, or offer human handoff based on your policy.

02 Patterns

Common production tools

Teams ship these tool bundles on day one:

  • crm.lookup_contact — hydrate caller context from Salesforce or HubSpot
  • calendar.book_slot — real-time Cal.com availability check and booking
  • payments.send_link — Razorpay/Stripe payment link over SMS post-call
  • kb.search — vector retrieval against attached knowledgebases
