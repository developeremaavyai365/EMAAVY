API Documentation

Batch & Queue APIs

Enqueue thousands of outbound dials with intelligent pacing, timezone windows, retry policies, and per-sub-account rate limits.

01 POST https://api.emaavy.com/v1/batches

Create a batch job

Upload contacts and schedule a campaign wave.

Example request

curl -X POST https://api.emaavy.com/v1/batches \
  -H "Authorization: Bearer <EMAAVY_API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{
    "agent_id": "agt_4kLm9",
    "name": "June renewal reminders",
    "contacts": [
      { "to": "+919876543210", "vars": { "first_name": "Anita", "due_date": "2026-06-15" } }
    ],
    "pacing": { "max_concurrent": 40, "calls_per_minute": 120 },
    "retry": { "no_answer": { "attempts": 2, "delay_minutes": 180 } }
  }'

Request body

{
  "agent_id": "agt_4kLm9",
  "name": "June renewal reminders",
  "contacts": [{ "to": "+919876543210", "vars": { "first_name": "Anita", "due_date": "2026-06-15" } }],
  "pacing": { "max_concurrent": 40, "calls_per_minute": 120 },
  "retry": { "no_answer": { "attempts": 2, "delay_minutes": 180 } }
}

Responses

200 — OK — batch queued
{
  "batch_id": "batch_7hK2",
  "status": "queued",
  "contact_count": 1,
  "estimated_start": "2026-05-30T11:00:00Z"
}
400 — Bad Request
{
  "error": {
    "code": "invalid_request",
    "message": "Missing required field or malformed JSON body.",
    "field": "agent_id"
  }
}
401 — Unauthorized
{
  "error": {
    "code": "unauthorized",
    "message": "Invalid or missing Authorization header. Use Bearer <EMAAVY_API_KEY>."
  }
}
501 — Not Implemented
{
  "error": {
    "code": "not_implemented",
    "message": "This endpoint is not enabled for your workspace tier. Contact sales@emaavy.com."
  }
}
