Overview

Real-Time Ears is the supervisor console for active calls. Every seat appears on a map with transcript preview, sentiment indicator, and intervention controls — built for BPO floors and inside sales pods alike.

What you see in the dashboard

Key views and signals available in this module.

  • Active call map view with campaign and agent attribution
  • Live transcript preview scrolling word-by-word
  • Sentiment color coding shifting green, amber, or red per turn
  • One-click supervisor join or whisper-coach without dropping the AI agent

How it works on EMAAVY

  1. 01Stream

    STT pipelines push partial transcripts over WebSocket.

  2. 02Score

    Sentiment model updates on each conversational turn.

  3. 03Display

    Supervisor UI renders captions with speaker labels.

  4. 04Intervene

    Authorized users bridge into the call with full context.

  5. 05Archive

    Completed streams merge into the searchable call record.

Typical use cases

  • Live QA on high-value sales calls
  • Escalation before a frustrated customer churns
  • Training new human agents with real examples
