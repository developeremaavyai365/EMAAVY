Overview

The leaderboard is not vanity metrics. It ties talk patterns to outcomes so workforce planners know which agents to replicate, which scripts to fix, and where AI is outperforming manual dialing.

What you see in the dashboard

Key views and signals available in this module.

  • Per-agent conversion rate with trend arrows
  • Average handle time vs team median
  • Coaching compliance score from script adherence
  • Side-by-side AI vs human performance bands

How it works on EMAAVY

  1. 01Attribute

    Calls map to operator or AI agent IDs.

  2. 02Score

    Outcomes, sentiment, and script tags roll into KPIs.

  3. 03Rank

    Leaderboard refreshes on a configurable interval.

  4. 04Coach

    AI-generated notes highlight specific improvement moments.

  5. 05Export

    HR and ops teams pull period summaries for reviews.

Typical use cases

  • Weekly team standups with objective rankings
  • Deciding which AI voice to clone for new campaigns
  • Compliance reviews tied to script adherence scores
